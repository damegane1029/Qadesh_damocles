
スライドショーシステム / SlideshowSystem 利用規約

-------------------
VRChatのワールドで使えるUdon製スライドショースクリプトと3Dモデルのセットです。
操作可能なプレゼンテーション用タブレットとモニターのPrefab、自動で画像が切り替わるデジタルフォトフレームのPrefabが含まれていて、
任意の画像を何枚でも設定できます。

※Udon製ギミックの為、使用するにはUnityにVRChatSDK3及びUdonSharpのインポートが必要です。

-------------------
【 必要動作環境 】
- VRCSDK3(WORLD)
https://vrchat.com/home/download
- UdonSharp
https://github.com/MerlinVR/UdonSharp/releases
それぞれ最新版のUnityPackageをUnityにインポートして下さい。

-------------------
【 内容物 】
UnityPackage形式(Unity2019.4.31fで作成)

■ Prefab関連
・ デジタルフォトフレーム 全4種 (White/Black/Pink/Wood)
・ プレゼンテーション用タブレット (スライド操作ギミック付き)
・ スライドショー表示モニター(自動スライド送り版)
・ スライドショー表示モニター(複製用) 2種 平面/湾曲

■ モデル関連
・ デジタルフォトフレーム3Dモデルfbx 1092△ポリゴン
・ プレゼンテーション用タブレット3Dモデルfbx 874△ポリゴン
・ スライドショー表示用モニター(平面)3Dモデルfbx 346△ポリゴン
・ スライドショー表示用モニター(湾曲)3Dモデルfbx 790△ポリゴン
(BaseColorテクスチャ / Metalicテクスチャ / NormalMapテクスチャ / Roughnessテクスチャ)

-------------------
【 使用方法 】
1. UnityPackageをUnityにインポートする。(先にVRCSDK3及びUdonSharpのインポートが必要です)
2. シーン内にPrefabを配置
3. 配置したPrefabに付いているSlideshowスクリプトの「スライド画像リスト」に任意の画像をドラッグ&ドロップする。
3. VRChatにワールドとしてアップロードしてVRChat上で動作を確認。

-------------------
【 使用上の注意 】
・ VRChatやUdonSharpの仕様変更により動作しなくなる場合があります。ご了承下さい。
・ 反射を綺麗に表現するには、ライティング及びReflection Probeの設定を行って下さい。
・ テクスチャ解像度が高めなので、コンパイル後のデータ容量を気にする場合は必要に応じてInspectorより画像の解像度の変更等を行って下さい。(デフォルトで圧縮設定済みです)
